# Cloud Security Program (AWS / Azure)

## Objective
Design and manage cloud security controls for a multi-account cloud environment
supporting distributed teams and remote operations.

## Scope
- Identity & Access Management (IAM)
- Centralized logging and monitoring
- Security posture management
- Threat detection and alerting

## Approach
- Implemented least-privilege access models
- Centralized audit logging for visibility
- Enabled continuous security monitoring
- Defined cloud security guardrails

## Outcome
- Improved security visibility
- Reduced misconfigurations
- Faster incident detection and response
